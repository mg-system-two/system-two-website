# How We Teach - Demo Page

This is a self-contained demo of the "How We Teach" page for the System Two website.

## What's included:

- `how-we-teach.html` - The complete page with three hero essays (Feynman, Halsted, Alcuin)
- `assets/styles.css` - All styles
- `assets/system2-logomark.png` - Logomark for header
- `assets/android-chrome-512x512.png` - Favicon

## To view:

Simply open `how-we-teach.html` in a browser.

## Notes:

- Illuminated capitals are currently placeholder `<div>` elements (styled letters)
- Replace these with actual image files when ready
- Navigation links point to other pages that don't exist in this demo (they're in the full site)
- The page uses Palatino Linotype font with proper fallbacks
